# Bot replay recorder

Probably not the best solution to the problem as you need to watch the whole replay to get the recording  
Recording does lead to a slight decrease in render framerate

Outputs the recording as a .webm   
The files it outputs are playable but are missing duration info (seems to be a known issue with MediaRecorder and can probably be fixed)

bot.bin is the bot file loaded
map.bin is the map file loaded

Demo: https://nekoify.github.io/bot-replay-recorder/
